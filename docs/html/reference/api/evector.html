Constructor#

class pyevspace.EVector([initializer])#

Create an EVector object with initial values from initializer, or zero if initializer is omitted.

Parameters:

initializer (iterable) – an iterable object of length three with numeric values

Raises:

  • TypeError – if initializer is not an iterable

  • TypeError – if a value of initializer is not a numeric type

  • ValueError – if initializer does not have a length of exactly three

Vector Operators#

EVector.mag()#

Returns:

Return type:

float

EVector.mag2()#

Computes the square of the magnitude of the vector. Use this instead of squaring the result of EVector.mag() to avoid rounding errors.

>>> vector = EVector((0.619286, 0.581799, 0.913961))
>>> mag = vector.mag()
>>> magSquared = vector.mag2()
>>> magSquared == (mag * mag)
False

This yields the same result as dot(self, self). See also (py:func:dot).

Returns:

the square of self.mag()

Return type:

float

EVector.normalize()#

Normalizes the vector by dividing each element by the magnitude of the vector. This results in a vector with lengh equal to one.

Returns:

None

Iterator Protocol#

The iterator protocol for the EVector type is not managed through a separate iterator type but within the class itself.

Return type:

Raises:

Buffer Protocol#

The EVector class supports the buffer protocol and can be used by other object which also support the buffer interface. For example it can be used to instantiate a memoryview object

>>> vector = EVector((1, 2, 3))
>>> view = memoryview(vector)
>>> view[2] = 3.14
>>> print(vector)
[1, 2, 3.14]
